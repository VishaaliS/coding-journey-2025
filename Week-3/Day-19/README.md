## What I Learned Today

### Queue Basics

**Concept:** FIFO (First In, First Out)

**Implementation with Arrays:**
- Enqueue operation (add to rear)
- Dequeue operation (remove from front)
- Front and rear pointer management
- Circular queue optimization
- Overflow and underflow handling

**Implementation with Linked List:**
- Enqueue at tail
- Dequeue from head
- Dynamic size (no fixed capacity)
- More flexible than array implementation

**Java Built-in Queue:**
- `Queue<Type> queue = new LinkedList<>();`
- `queue.offer(element)` - add to rear
- `queue.poll()` - remove from front
- `queue.peek()` - view front
- `queue.isEmpty()` - check if empty

### Implement Queue using Stacks
**LeetCode 232** - [Link](https://leetcode.com/problems/implement-queue-using-stacks/)
- Pattern: Two stacks to simulate queue behavior
- Difficulty: Easy
- Approach: 
  - Stack 1 for enqueue
  - Stack 2 for dequeue
  - Transfer elements when dequeue stack is empty
- This simulates FIFO using two LIFO structures
